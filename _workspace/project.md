# Project — gennyoon.net

> The "why" — high-level intent, audience, and constraints.
> This file rarely changes. Roadmap and Sprint files do the moving.

## What

전직 CTO GennYoon의 AI 노마드 블로그 — React 19 + Vite 6 + Supabase

## Why

AI 시대의 노마드 개발 여정을 기록하고, LinkedIn / Medium / Dev.to로 크로스포스팅하여 개인 브랜드를 구축한다.

## Constraints

- GLM-5 AI API (bigmodel.cn) 의존 — 중국 서비스 특성상 지연/가용성 변동 있을 수 있음.
- 크로스포스팅 토큰은 만료/재발급 주기 관리 필요.

## Non-goals

- 멀티 사용자 / 팀 블로그 플랫폼 아님 (단일 저자).
- 댓글/소셜 기능 (외부 플랫폼 위임).
